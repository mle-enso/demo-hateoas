demo-hateoas

Prototype for showing strongly typed HATEOAS links in conjunction with so-called page type mappings.

Build

After cloning the repository execute mvn clean package to run the example test which also serves for generating the documentation with Spring REST Docs.

You now can take further information from /target/generated-docs/index.html .
